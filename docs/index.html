Kebijakan Privasi OmsetKu

Terakhir diperbarui: 30 Mei 2026

Versi 3.8.0+

📌 Tentang OmsetKu

OmsetKu adalah aplikasi pencatatan omset untuk UMKM Indonesia. Kami berkomitmen melindungi privasi pengguna dan data bisnis Anda.

✅ Semua data tersimpan lokal di HP Anda — tidak ada server, tidak ada cloud wajib.

📥 Data yang Dikumpulkan

OmsetKu hanya menyimpan data yang Anda input sendiri:

  • Nama bisnis dan nama sales / karyawan
  • Data transaksi: nomor bon, tanggal, nama pelanggan, nominal, catatan
  • Pengaturan aplikasi (format bon, tema, preferensi)

Yang TIDAK kami kumpulkan:

  • Identitas pribadi (nama, email, nomor HP Anda)
  • Lokasi / GPS
  • Kontak dari perangkat
  • Data pembayaran / kartu kredit
  • Informasi biometrik

💾 Penyimpanan Data

PropertiKeterangan
LokasiStorage internal HP (SQLite lokal)
AksesHanya app OmsetKu yang bisa membaca
InternetTidak diperlukan — 100% offline
ServerTidak ada pengiriman ke server kami

☁️ Backup Google Drive (Opsional)

Fitur backup Google Drive bersifat opsional. Jika diaktifkan:

  • File backup diunggah ke folder "OmsetKu Backup" di Google Drive Anda
  • Hanya akun Google yang Anda pilih yang dapat mengakses file tersebut
  • Kami tidak dapat mengakses Google Drive Anda tanpa izin eksplisit
  • Anda dapat mencabut izin kapan saja melalui Pengaturan Google

📂 Izin Aplikasi

IzinKeperluan
READ_EXTERNAL_STORAGEMembaca file CSV untuk import data
WRITE_EXTERNAL_STORAGEMenyimpan file export Excel / backup
USE_BIOMETRICFingerprint lock (opsional)
RECEIVE_BOOT_COMPLETEDBackup otomatis harian (jika aktif)
InternetHanya untuk backup Google Drive (opsional)

🗑️ Hak Pengguna

Karena data tersimpan lokal di perangkat Anda, Anda memiliki kendali penuh:

  • Hapus data: Settings → Reset Semua Data
  • Export data: Settings → Export Excel / Export Backup
  • Hapus semua: Uninstall aplikasi menghapus seluruh data
