Kebijakan Privasi Omset Pintar

Terakhir diperbarui: 3 Juni 2026

Versi 1.0.0+

📌 Tentang Omset Pintar

Omset Pintar adalah aplikasi pencatatan omset harian untuk UMKM Indonesia. Kami berkomitmen melindungi privasi pengguna dan data bisnis Anda.

✅ Semua data tersimpan lokal di HP Anda — tidak ada server Omset Pintar yang menerima data bisnis Anda.

📥 Data yang Dikumpulkan

Omset Pintar hanya menyimpan data yang Anda input sendiri:

  • Nama bisnis dan nama sales / karyawan
  • Data transaksi: nomor bon, tanggal, nama pelanggan, nominal, catatan
  • Pengaturan aplikasi (format bon, tema, preferensi, jam notifikasi)

Jika fitur Google Drive diaktifkan (opsional):

  • Email akun Google Anda (disimpan lokal, terenkripsi)
  • Token OAuth (disimpan lokal di SecureStore terenkripsi)

Yang TIDAK pernah kami kumpulkan:

  • Identitas pribadi (selain email Google jika Drive aktif)
  • Lokasi / GPS
  • Kontak dari perangkat
  • Data biometrik (fingerprint hanya untuk verifikasi lokal)
  • Analytics / telemetri penggunaan
  • Data pembayaran / kartu kredit

☁️ Google Drive — Backup & Sinkronisasi (Opsional)

Fitur Google Drive sepenuhnya opsional. Jika diaktifkan:

  • File backup diunggah ke folder "Omset Pintar Backup" di Google Drive Anda sendiri
  • Sinkronisasi antar 2 HP menggunakan Drive sebagai perantara — bukan server Omset Pintar
  • Koneksi menggunakan OAuth 2.0 + PKCE (aman, tanpa client secret)
  • Token diperbarui otomatis — tidak perlu login ulang berulang
  • Anda dapat mencabut izin kapan saja di: myaccount.google.com → Keamanan

🔒 Keamanan

  • Data lokal dilindungi enkripsi storage Android
  • Token OAuth disimpan di Android SecureStore (terenkripsi)
  • Kunci aplikasi fingerprint — lock otomatis saat app ditutup
  • Reset data memerlukan konfirmasi ganda + verifikasi fingerprint (jika aktif)

🗑️ Hak Pengguna

  • Hapus semua data: Settings → Reset Semua Data (konfirmasi ganda)
  • Export data: Settings → Laporan (Excel) atau Export Backup JSON
  • Putus Drive: Settings → Backup Google Drive → Putus Koneksi
  • Hapus total: Uninstall aplikasi menghapus seluruh data lokal
  • Cabut akses Google: myaccount.google.com → Keamanan → Aplikasi pihak ketiga

👦 Data Anak-Anak

Aplikasi ini ditujukan untuk penggunaan bisnis oleh orang dewasa. Kami tidak mengumpulkan data dari anak-anak di bawah usia 13 tahun.
