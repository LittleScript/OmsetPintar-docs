## 📄 FULL DESCRIPTION (max 4000 karakter)

```
Omset Pintar — Aplikasi Catat Omset untuk Toko & UMKM Indonesia

Kelola omset penjualan harian toko atau UMKM Anda dengan mudah dan cepat. Dirancang khusus untuk pemilik toko yang memiliki 1–20 sales, Omset Pintar membantu Anda mencatat setiap transaksi, memantau performa sales, dan menghasilkan laporan profesional — semuanya dari genggaman tangan.

✅ FITUR UTAMA

📊 Dashboard Real-Time
Pantau omset hari ini, minggu ini, dan bulan ini sekaligus. Lihat perbandingan % naik/turun vs periode sebelumnya. Grafik garis harian per sales, ranking pelanggan, dan hari tersibuk tersaji dalam satu layar.

⚡ Input Transaksi Super Cepat
Nomor bon auto-generate, autocomplete nama pelanggan 3 level, haptic feedback saat simpan. Input satu transaksi dalam hitungan detik. Tombol ‹ › untuk navigasi bon dan tanggal.

💱 Multi Currency
Dukung 6 mata uang: IDR (Rp), USD ($), MYR (RM), SGD (S$), EUR (€), GBP (£). Ganti mata uang kapan saja di Settings — semua tampilan menyesuaikan otomatis.

🌐 Bilingual Indonesia / English
Toggle bahasa instan di header app. Semua teks berubah sekaligus — cocok untuk pengguna internasional.

📈 Ranking & Analitik Pelanggan
Siapa pelanggan terbesar bulan ini? Siapa yang paling sering beli? Ranking otomatis per sales, per periode (hari/minggu/bulan/tahun).

📊 Laporan Excel Profesional
Export ke Excel (.xlsx) dengan styling lengkap: ringkasan KPI, rekap per sales, rekap bulanan, daftar pelanggan, ranking. Pilih periode (tahun atau bulan tertentu). Siap cetak atau kirim ke atasan.

☁️ Backup & Sinkron Google Drive
Backup otomatis ke Google Drive Anda. Sinkronisasi 2 HP untuk tim dengan sales berbeda — data langsung tergabung tanpa ribet.

🔍 Deteksi Typo Nama Otomatis
Algoritma Jaro-Winkler mendeteksi nama pelanggan yang mirip atau typo. Gabungkan otomatis semua transaksi ke nama yang benar.

📤 Share Rekap sebagai Gambar
Bagikan rekap omset harian, mingguan, atau bulanan sebagai kartu gambar langsung ke WhatsApp atau media sosial — bebas pilih tanggal.

🔒 Aman & Privat + Kunci Fingerprint
Semua data tersimpan lokal di HP Anda. Kunci aplikasi dengan fingerprint/PIN — app terkunci otomatis setelah 30 detik di background (seperti app bank). Tidak ada server yang menyimpan data bisnis Anda.

📱 Widget Home Screen *(coming soon)*
Lihat omset hari ini langsung dari layar utama tanpa membuka aplikasi.

---

COCOK UNTUK:
• Toko retail, grosir, minimarket
• UMKM dengan 1–20 sales
• Bisnis dengan 10–500 transaksi per hari
• Pemilik usaha yang butuh laporan cepat
• Bisnis internasional (multi currency)

---

TIDAK BUTUH INTERNET untuk penggunaan sehari-hari. Internet hanya diperlukan untuk backup Google Drive (opsional).

Data Anda 100% milik Anda — tidak ada iklan, tidak ada tracking, tidak ada pengiriman data ke server kami.
```

## 📊 DATA SAFETY FORM (Play Console)

| Pertanyaan | Jawaban |
|---|---|
| Apakah app mengumpulkan/berbagi data pengguna? | **Ya** (kondisional) |
| Data apa yang dikumpulkan? | Email (hanya jika Drive aktif), preferensi bahasa & mata uang (lokal) |
| Apakah data dienkripsi? | Ya |
| Apakah pengguna bisa request hapus data? | Ya (Reset + uninstall) |
| Apakah ada data sharing ke pihak ketiga? | Tidak |

**Tipe data yang dikumpulkan:**
- ☁️ App activity: Tidak
- 💳 Financial info: Tidak
- 📍 Location: Tidak
- 📧 Personal info: Email (jika Google Drive aktif, tersimpan lokal)
- 🔐 Health & fitness: Tidak
- 📱 Device info: Tidak
- 📁 Files & Storage: **Ya** — file CSV/JSON export disimpan di `Downloads/OmsetPintar/` di HP user; gambar rekap disimpan ke Galeri. **Tidak dikirim ke server manapun.**

**Izin Penyimpanan (Storage Permission):**
| Izin | Tujuan | Wajib? |
|---|---|---|
| `READ_EXTERNAL_STORAGE` | Baca file saat import CSV/backup JSON | Tidak (hanya saat import) |
| `WRITE_EXTERNAL_STORAGE` | Simpan file export CSV/JSON ke Downloads/OmsetPintar/ | Tidak (hanya saat export) |
| `READ_MEDIA_IMAGES` | Simpan gambar rekap ke Galeri HP (Android 13+) | Tidak (hanya saat Save to Gallery) |

**Penegasan:** Semua file tersimpan 100% di HP pengguna. Tidak ada data yang diunggah ke server Omset Pintar. Tidak digunakan untuk tracking, analytics, atau keperluan lain di luar yang tercantum.
